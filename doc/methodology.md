# Methodology

## Approach

The project follows a structured data analysis workflow:

1. Data Exploration  
2. Data Cleaning  
3. Data Transformation  
4. Analysis  
5. Validation  
6. Reporting  

---

## Data Exploration

- Inspected table structures  
- Verified relationships between tables  
- Identified missing and inconsistent values  

---

## Data Cleaning

- Corrected misclassified pollution records  
- Removed duplicate visit entries  
- Standardized text values  

---

## Data Transformation

- Aggregated population by water source  
- Calculated average queue times  
- Created ranking and summary tables  

---

## Analysis

- Performed joins across multiple tables  
- Used aggregation functions for insights  
- Applied window functions for ranking  

---

## Data Validation

- Compared survey data with auditor dataset  
- Identified mismatched records  
- Calculated employee error rates  

---

## Output

- Built analytical views  
- Created summary tables  
- Designed project tracking structure
